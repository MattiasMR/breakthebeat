# Break The Beat Site

Sitio estatico en Astro para `Break The Beat 3ra edicion`.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El output queda en `dist/`.

## Cloudflare Pages

- Framework preset: `Astro`
- Build command: `npm run build`
- Output directory: `dist`
- Produccion temporal: subdominio `.pages.dev`

## Edicion de contenido

El contenido principal esta en `src/data/site.ts`.

Los canales de pago estan creados como estructura, pero todos quedan desactivados hasta confirmar cuenta responsable, links oficiales, politica de devolucion y rendicion.

## Notas de marca

El logo oficial optimizado para la web esta en `public/assets/break-the-beat-logo.webp`; el favicon esta en `public/assets/favicon.png`.

Las fuentes Aurochs existen en los materiales de marca, pero no se self-hostearon en esta version hasta confirmar licencia de uso web.
